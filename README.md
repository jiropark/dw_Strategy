# KOSPI 100 Trading Strategy Simulator

KOSPI 시가총액 상위 100종목 대상 6개 기술적 분석 전략 기반 트레이딩 시뮬레이션 시스템

## 개요

3년 8개월치(2022-07 ~ 2026-03, 900거래일) KOSPI 데이터를 활용하여 6개 기술적 전략의 성과를 백테스트하고, 500만원 투자 시 예상 수익을 시뮬레이션합니다.

## 전략 설명

| 전략 | 로직 | 매수 조건 |
|------|------|-----------|
| MA 교차 | 5일/20일 이동평균선 교차 | 골든크로스 (5일 > 20일) |
| MACD | 12/26/9 MACD | MACD > Signal Line |
| RSI | 14일 RSI | RSI < 30 (과매도) |
| 볼린저밴드 | 20일, 2표준편차 | 종가 < 하단밴드 |
| 모멘텀 | 5일/20일 수익률 | 단기+장기 모두 상승 |
| 거래량 | 20일 평균 대비 | 150%+ 거래량 + 가격 상승 |

## 리스크 관리

- **Stop Loss**: -1% 도달 시 즉시 청산
- **Take Profit**: +0.3% 도달 시 즉시 청산
- **종목당 최대 비중**: 10%
- **연속 3회 손실 시**: 거래 중단 후 재개
- **거래비용**: 0.3% (수수료 + 슬리피지)
- **보유기간**: 3거래일

## 시장 국면 감지

KOSPI 20일 누적 수익률 기반 자동 분류:
- **상승장**: 20일 수익률 > +2%
- **하락장**: 20일 수익률 < -2%
- **횡보장**: 그 외

국면별로 전략 가중치가 자동 조정됩니다.

## 핵심 인사이트

1. **하락장에서 가장 효과적** — RSI 과매도 + 볼린저밴드 하단 이탈 시 반등 포착이 핵심
2. **앙상블 전략이 가장 안정적** — MDD 7.3% (KOSPI 보유 20.7% 대비)
3. **개별 전략 중 거래량 전략이 수익률 1위** — 단, MDD 18%로 변동성 큼
4. **모든 전략이 KOSPI 단순보유(+140%) 대비 초과 수익 달성**

## 주의사항

- 이 시뮬레이션은 과거 데이터 기반 백테스트이며, 미래 수익을 보장하지 않습니다
- 실제 매매 시 슬리피지, 호가 스프레드 등 추가 비용이 발생할 수 있습니다
- 투자 판단은 본인의 책임 하에 이루어져야 합니다
